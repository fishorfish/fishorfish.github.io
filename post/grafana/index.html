    Первая попытка настроить дашборд.

    Для вывода на дашборд nginx-логов используется инструмент Grafana.

    В докеризированном проекте для вывода конкретного сервиса используется Query - appname:“some_name”, в недокеризированном - к файлу с логами.

    В пункте Alias можно вписать название алиаса, который будет ссылаться на наш Query в графе Terms, и затем по алиасу можно настраивать/фильтровать вывод результатов.

    Также можно вывести логи самого приложения - php контейнера.
