    • Codeception: подключение к TestRail

    Sat, Aug 15, 2020 1-minute read

    Для подключения Codeception к TestRail:

    1. Выполнить команду composer require --dev bookit/codeception-testrail

    2. В конфиге codeception.yml добавить настройки подключения к TestRail

    3. Поменять в composer.json установленную версию "bookit/codeception-testrail" на "dev-master" → composer update bookit/codeception-testrail

    4. Создать в корне проекта файл .env.test и добавить в него креды от TestRail

    5. В случае, если composer.json не сконфигурировал автоматически autoload-dev”, то вручную сконфигурировать в composer.json: "autoload-dev", добавив в "psr-4": "Your_namespace\\Tests\\": "tests/” → composer dump-autoload

    6. Проверить наличие корректных неймспейсов во всех файлах папки tests

    7. Для вывода ссылки (при прогоне тестов) с данными о прохождении автотестов создать папку tests/Extensions, создать в ней файл TestRailExtension.php

    8. Для слинковки тестов с тест-кейсами в TestRail над каждым методом добавлять аннотации: @tr-suite ХХХХ (id тест-сьюта) и @tr-case ХХХХ (id тест-кейса)

    9. Войти в контейнер и запустить тест командой vendor/bin/codecept run

    10. Проверить отображение результатов прогона тестов в TestRail, вкладка Test Runs & Results
